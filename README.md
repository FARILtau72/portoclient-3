# Portfolio Khaidar Ali

Website portfolio personal dengan tema kartu game (Yu-Gi-Oh style) dan animasi interaktif.

## Ringkasan
Portfolio ini dibuat sebagai landing page personal untuk menampilkan profil, skill, work history, dan 1 project utama.

## Fitur Utama
- Hero section dengan hanging card interaktif
- Flip card 3D di desktop (hover) dan mobile (tap)
- Animasi pendulum/ayunan pada kartu samping
- Layout responsive untuk mobile, tablet, dan desktop
- Section About, Work History, Projects, dan Contact
- 1 project utama (live project)

## Tech Stack
- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Lucide Icons
- Google Fonts

## Struktur Folder
- index.html: halaman utama
- README.md: dokumentasi proyek

## Menjalankan Proyek
Tidak perlu build tools.

Cara paling cepat:
1. Buka file index.html langsung di browser.
2. Atau jalankan Live Server dari VS Code agar lebih nyaman saat edit.

## Catatan
Jika mau migrasi ke React atau Next.js, struktur konten saat ini sudah siap dipisah menjadi komponen per section.
